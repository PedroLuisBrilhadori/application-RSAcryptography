Um projeto usando [Next.js](https://nextjs.org/)

## Getting Started

Primeiro, é necessário baixar as dependências:

```bash
npm install
# or
yarn
```

Para rodar o projeto, utilize o comando:

```bash
npm run dev
# or
yarn dev
```

Abra o navegador no link: [http://localhost:3000](http://localhost:3000)
